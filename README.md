# Projeto SQL – Clínica de Fisioterapia
Este repositório contém todos os scripts SQL desenvolvidos para a
Experiência Prática IV – Implementação e Manipulação de Dados.

## Arquivos incluídos:

✔ 01_create_tables.sql  
✔ 02_insert_data.sql  
✔ 03_queries.sql  
✔ 04_updates_deletes.sql  

## Objetivo
Implementar e manipular dados no banco da Clínica de Fisioterapia,
utilizando comandos SQL de criação, inserção, consulta, atualização
e exclusão de dados.

## Tecnologias utilizadas
- MySQL Workbench (ou PGAdmin)
- SQL padrão
